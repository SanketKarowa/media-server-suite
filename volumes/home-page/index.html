Multimedia Server Suite

Welcome to your personal media server suite dashboard which is made to fulfill all your multimedia requirements stacked at one place. It comprises Sonarr, Radarr, Jellyfin, qBitorrent, Jackett and h5ai to provide you rich multimedia experience.

Easy to setup

With the combination of various docker containers, it's easy to deploy the applications in one place. All the application runs in a separate docker containers which can be easily managed by Portainer. Just run the provided YAML file with docker-compose and rest will be taken care by it.

About

Benefits of this

This stack has been created to solve the common problems that we face when we want to watch, stream or download our favorite TV series or movies. Also, it's very cumbersome to manually install and manage differrent applications. Therefore, with the power of docker containers those problems are the things of past.

  • Includes all necessary media packages
  • Provides minimum and hasslefree setup
  • Automates the build and deployment

Just add your favorite TV series to Sonarr and movies to Radarr and they will handle the rest of things like fetching the appropriate files and downloading them with the help of qBittorrent. Once the file has been downloaded, then you can stream it using Jellyfin that provides powerful interface for media consumption.

Services

What it contains

Portainer

Portainer is a universal container management tool. It works with Kubernetes, Docker, Docker Swarm and Azure ACI and allows you to manage containers without needing to know platform-specific code.

Sonarr

Sonarr is a PVR for Usenet and BitTorrent users. It can monitor multiple RSS feeds for new episodes of your favorite shows and will grab, sort and rename them.

Radarr

Radarr is a movie collection manager for Usenet and BitTorrent users. It can monitor multiple RSS feeds for new movies and will interface with clients and indexers to grab, sort, and rename them.

Jellyfin

Jellyfin is the volunteer-built media solution that puts you in control of your media. Stream to any device from your own server, with no strings attached.

qBittorrent

The qBittorrent project aims to provide an open-source software alternative to µTorrent. It is based on the Qt toolkit and libtorrent-rasterbar library.

h5ai

h5ai is a modern file indexer for HTTP web servers with focus on your files. Directories are displayed in a appealing way and browsing them is enhanced by different views, a breadcrumb and a tree overview.

Jackett

Jackett works as a proxy server: it translates queries from apps (Sonarr, Radarr, etc.) into tracker-site-specific http queries, parses the html response, then sends results back to the requesting software.

FlareSolverr

FlareSolverr starts a proxy server and it waits for user requests in an idle state using few resources. When some request arrives, it uses puppeteer with the stealth plugin to create a headless browser (Chrome).
